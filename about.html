About LimeTech

LimeTech is a fictional tech brand created to showcase a fully functional website for computer products. The goal of this project is to design a clean, responsive, and modern landing page where users can explore a variety of computer hardware like CPUs, gaming PCs, accessories, and more.

This website includes interactive sections such as animated scroll effects, smooth navigation, and product imagery. It aims to demonstrate frontend design skills using HTML and CSS, with potential for future expansion using JavaScript and database integration.

Every section is designed to highlight modern tech visuals, providing users with a smooth and immersive browsing experience.
